# WooCommerce Product Filter by Titan

A powerful and flexible WooCommerce product filtering plugin with SEO-friendly URLs, instant filtering, and advanced customization options.

---

## 🚀 Features

- **SEO-Friendly URLs**  
  Generate clean and readable filter URLs like:  
  `/filters/product_brand-acer`  
  Improves both SEO performance and user experience.

- **Advanced SEO Support**  
  - Flexible URL structure (Slug or ID)
  - Custom Meta Robots for filtered pages
  - Canonical URLs for filters
  - Custom SEO Titles for better ranking

- **Instant Filtering (AJAX)**  
  Apply filters instantly on click or use an "Apply" button based on your preference.

- **Drag-and-Drop Builder**  
  Easily organize attributes and terms directly from the admin panel.

- **Term Images Support**  
  Upload images for filter terms (e.g., brand logos).

- **Custom Price Ranges**  
  Define flexible price ranges tailored to your store.

- **Product Counts Display**  
  Show how many products match each filter (e.g., *Acer (594)*).

- **Hide Empty Terms**  
  Automatically hide filter options with no available products.

- **Custom Text Support**  
  Easily customize filter labels, headings, and UI text.

- **Custom Styling (CSS)**  
  Fully customize the appearance of your filters.

- **Cache Support**  
  Improve performance with built-in caching compatibility.

- **Mobile-Friendly Design**  
  Responsive filter interface with selected filter tags optimized for mobile users.

- **Widget & Layout Support**  
  Display filters as widgets or horizontal filter bars.

- **Category & Brand Assignment**  
  Assign filters to specific categories, brand pages, or the shop page.

- **And More...**  
  Additional powerful features waiting for you to explore 🚀

---

## 📦 Installation

1. Upload the plugin to the `/wp-content/plugins/` directory  
   **or** install it directly via the WordPress admin panel.
2. Activate the plugin from the **Plugins** menu.
3. Navigate to:  **Product Filter**
4. Configure your filters and assign them as needed.
5. Go to **Settings → Permalinks** and click **Save Changes** to flush and reset WordPress rewrite rules.
---

## ⚙️ Usage

1. Create filters in the admin panel by selecting attributes such as:
   - Price
   - Brand
   - CPU
   - Memory
   - Custom taxonomies

2. Assign filters to:
   - Product categories
   - Brand pages
   - Shop page

3. Customize the appearance using built-in CSS and text settings.

4. Choose filtering behavior:
   - Instant filtering
   - Apply button

5. Enable optional features:
   - Product counts
   - Hide empty terms

## ❓ FAQ

### Does it support custom taxonomies?
Yes. You can create filters for any WooCommerce taxonomy, including custom ones.

### Can I customize the filter appearance?
Absolutely. You can use custom CSS and text settings in the admin panel.

### Is it SEO-friendly?
Yes. The plugin generates clean, structured URLs optimized for search engines.
