A powerful WooCommerce product filter with SEO-friendly URLs, instant filtering, and highly customizable attributes.

== Description ==
WC Product Filter enhances your WooCommerce store with advanced product filtering. Create filters for price, brand, CPU, memory, and more, with SEO-friendly URLs (e.g., `/filters/product_brand-acer`). Features instant filtering, drag-and-drop attributes, term images, and custom CSS for a seamless user experience.

== Key Features ==
- **SEO-Friendly URLs**: Beautiful filter URLs like `/filters/product_brand-acer` for better SEO and user experience.
- **Instant Filtering**: Apply filters instantly on click or via an apply button.
- **Drag-and-Drop Attributes**: Easily arrange filter attributes and terms in the admin.
- **Term Images**: Upload images for terms (e.g., brand logos).
- **Custom Price Ranges**: Create custom price ranges for flexible filtering.
- **Display Product Counts**: Show the number of products per term (e.g., Acer (594)).
- **Hide Empty Terms**: Hide terms with no products in the frontend filter.
- **Custom CSS and Text**: Fully customize filter appearance and text.
- **Mobile-Friendly**: Responsive filter menu and selected filter tags for mobile users.
- **Widget Support**: Add filters as widgets or horizontal layouts.
- **Category and Brand Pages**: Assign filters to specific categories or brand pages.

== Installation ==
1. Upload the plugin to the `/wp-content/plugins/` directory or install via the WordPress admin.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to **WooCommerce > Settings > Product Filter** to configure filters, assign to categories, or customize settings.

== Usage ==
1. Create filters in the admin, selecting attributes like price, brand, or custom taxonomies.
2. Assign filters to categories, brand pages, or the shop page.
3. Customize filter appearance with CSS and text in the settings.
4. Enable instant filtering or button-based apply in the general settings.
5. Optionally display product counts or hide empty terms.
